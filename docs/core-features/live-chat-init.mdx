# Live Chat Initialization

Start a live chat session with your users.

## Updated Usage

Now, you must pass a config object:

```js
sdk.initChat({
  token: 'YOUR_API_TOKEN',
  userId: 'USER_ID', // Required
  userEmail: 'user@example.com', // Optional for enhanced identification
});
```

- `token`: Required API token
- `userId`: Required user identifier for tracking
- `userEmail`: Optional user email for enhanced identification

### Example Use Case
- Start a chat for a logged-in user with their ID

### Edge Cases
- If `userId` is missing, initialization will fail as it is a required parameter.
- Passing a string instead of an object will throw an error
